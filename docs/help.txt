帮助
此文档包括所有命令及功能的概述
1.界面
如你所见,这些界面上的语句都带有颜色,这些颜色代表了不同语句的类型
红色-描述
黄色-提示
绿色-输入符号
蓝色-用户输入的文字
2.交互
你可以在">>"后根据提示输入命令;若输入错误,你需要重新输入
3.命令
以下列举了目前有的一些命令和相对应的功能(这里只显示0级命令,1级和2级请在实操中查看)
exit                  退出
fIle                  对文件进行操作
help                  获取帮助内容
practice              练习使用
test                  测试
tool                  一些工具
update                检查更新
4.自定义命令
若你认为默认的命令有局限性,请在"user_custom.py"中自定义属于你自己的命令
定义方法:
on "user_custom.py":
...
def 命令名称():
    代码
...
on "cmds.py":
...
elif input_zz in ["命令名称"]:
    z_user.命令名称()
...
4.插件
由于技术限制,你只能将插件粘贴至"user_custom.py"中并在"cmds.py"中定义它
定义的方式与自定义命令的方式相同
5.反馈
若你在使用时遇到问题影响使用,请在"https://github.com/5h9igzqanx/zx-MyCdss/issues"中反馈给我们,我们会以最快的速度为你解惑
帮助文档完,感谢你的使用
